OBJETIVOS GENERALES:


Aprender a diseñar páginas web bajo HTML5 y CSS3


Objetivos Específicos:


  • Aprender a usar diferentes herramientas para maquetación web
  • Aprender sobre los diferentes elementos de secciones
  • Conocer las diferencias entre los diferentes elementos de secciones
  • Apredner a desarrollar sitios web en formato HTML5 y CSS3, para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad, por medio del uso de diferentes editores de código propuestos en el desarrollo del curso.
  • Diferenciar el uso correcto de las diferentes etiquetas de seccion (Header, Nav, Section, Article, Aside, Footer y Div.
  • Aprender a usar las diferentes propiedades de los estilos de CS3

Temáticas a desarrollar:


Introducción a HTML5, Estructura básica de HTML5, Aplicación de elementos de HTML5 a un proyecto de diseño Web, Introducción a CSS3, Estructura básica de CSS3, Aplicación de elementos de CSS3 a un proyecto de diseño Web.
